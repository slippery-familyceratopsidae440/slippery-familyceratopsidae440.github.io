## ❓ Frequently asked questions

**Do I need the original game files?**
Yes. Place your original game data files into the folder where you installed cnc-port. The engine detects these assets when it launches.

**Can I save my progress?**
The game creates a save folder in your documents directory. You can load and save games through the in-game menu.

**Performance is slow. What should I do?**
Check your graphics settings within the game. Lowering the resolution often improves performance on older hardware. Ensure your computer drivers are current.

**How do I update the game?**
Check the release link periodically for new versions. Download the new installer and run it over your existing folder to update the files.

**Is this an official release?**
This is an unofficial project. It relies on open source technology to improve game compatibility on modern operating systems.

## 🔧 Troubleshooting

If the game fails to launch, verify that you installed the latest version of the DirectX runtime from Microsoft. If you encounter errors, check the logs folder inside your installation directory. These text files explain errors during startup.
